# 🤖 AI Driven Customer Analytics Dashboard
# Web Link
 https://project-1-ai-driven-customer-6.onrender.com

A full-stack Flask-based AI analytics platform for customer segmentation, churn prediction, and recommendation insights using Machine Learning.

---

## 🚀 Features

### 📊 Dashboard
- Total Customers KPI
- Revenue Analytics
- Gender, Age, Income, Spending distribution charts
- Segment & Churn insights

### 👥 Customer Analytics
- Interactive data tables
- Multiple distribution charts
- Purchase behavior analysis

### 🧠 AI Features
- KMeans Clustering (Customer Segmentation)
- Random Forest (Customer Classification)
- Churn Risk Detection

### 🔮 Prediction System
- Predict Premium / Regular customers
- ML-based classification model

### 🎯 Recommendation Engine
- Product recommendations based on segment

### 📉 Churn Analysis
- High / Medium / Low risk customers
- Churn rate calculation

### 📄 PDF Report
- Download analytics report in PDF format
